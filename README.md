# 📖 Sobre o Projeto

O **Quiz para Programadores** é uma aplicação web desenvolvida com o objetivo de testar conhecimentos em programação por meio de perguntas interativas. O projeto foi criado como atividade prática para aplicar conceitos fundamentais do desenvolvimento Front-End, explorando recursos do HTML, CSS e JavaScript.

A proposta é oferecer uma experiência simples, dinâmica e intuitiva, permitindo que os usuários respondam perguntas relacionadas a tecnologias, conceitos de programação e desenvolvimento web, recebendo feedback ao final da atividade.

Além do aspecto educacional, este projeto também demonstra a utilização de recursos de interação com o usuário, validação de respostas e manipulação de elementos da página utilizando JavaScript.

---

# 🎯 Objetivos

Este projeto foi desenvolvido com os seguintes objetivos:

* Aplicar conceitos de desenvolvimento Front-End;
* Praticar a manipulação do DOM utilizando JavaScript;
* Desenvolver uma interface intuitiva e responsiva;
* Estimular o aprendizado por meio da gamificação;
* Exercitar lógica de programação e validação de dados;
* Consolidar conhecimentos adquiridos durante a disciplina.

---

# ✨ Funcionalidades

O sistema oferece funcionalidades como:

* Questionário interativo;
* Perguntas de múltipla escolha;
* Campos para respostas textuais;
* Validação das respostas;
* Exibição da pontuação final;
* Feedback sobre o desempenho do usuário;
* Interface organizada e de fácil navegação.

# ▶️ Como Executar

### 1. Clone este repositório

```bash

```

### 2. Entre na pasta do projeto

```bash
cd Quiz-Para-Programadores
```

### 3. Execute

Abra o arquivo **index.html** em seu navegador ou utilize a extensão **Live Server** do Visual Studio Code.
